Shapley values and axioms¶

The Shapley value is the classical fair-division solution for cooperative games. In explainability it underpins SHAP-style feature attributions when coalition payoffs are aggregated with Shapley weights. This page states the definition, the four defining axioms, and the uniqueness result from Shapley (1953).

For Banzhaf indices and estimand comparison in ML pipelines, see Estimands: Shapley vs Banzhaf. For business deployment contexts, see Business and compliance applications.

Definition¶

For a cooperative game with player set \(N\), \(|N| = n\), and characteristic function \(v\), the Shapley value \(\phi_i(v)\) for player \(i \in N\) is:

\[\phi_i(v) = \sum_{S \subseteq N \setminus \{i\}} \frac{|S|!\,(n-|S|-1)!}{n!} \bigl(v(S \cup \{i\}) - v(S)\bigr)\]

Interpretation: imagine players join a coalition in random order, uniformly over all \(n!\) permutations. The Shapley value is player \(i\)’s expected marginal contribution when joining.

For coalition size \(k = |S|\) (excluding \(i\)), the weight is:

\[w_{\mathrm{Shapley}}(k, n) = \frac{k!\,(n-k-1)!}{n!}\]

Implemented in ShapleyAggregator.

The four axioms¶

Shapley (1953) characterized \(\phi\) as the unique allocation rule satisfying four axioms on the space of cooperative games. Let \(\phi(v) = (\phi_1(v), \dots, \phi_n(v))\) denote the allocation vector for game \(v\).

Efficiency¶

The entire worth of the grand coalition is distributed:

\[\sum_{i \in N} \phi_i(v) = v(N)\]

Explainability reading: attributions sum to the model utility difference between the grand coalition and the empty baseline (when \(v(\emptyset)\) is the reference). This supports global consistency checks in audit workflows.

Symmetry¶

If players \(i\) and \(j\) make identical marginal contributions in every coalition — that is, \(v(S \cup \{i\}) = v(S \cup \{j\})\) for all \(S \subseteq N \setminus \{i, j\}\) — then:

\[\phi_i(v) = \phi_j(v)\]

Explainability reading: interchangeable tokens (e.g. duplicate padding) receive equal credit.

Dummy player (null player)¶

If player \(i\) never changes the payoff — formally \(v(S \cup \{i\}) = v(S)\) for all \(S \subseteq N \setminus \{i\}\) — then:

\[\phi_i(v) = 0\]

Explainability reading: tokens that do not affect the scored output receive zero attribution.

Additivity¶

For two games \(v\) and \(w\) on the same player set, define \((v + w)(S) = v(S) + w(S)\). Then:

\[\phi_i(v + w) = \phi_i(v) + \phi_i(w)\]

Explainability reading: attributions decompose additively when utilities are sums of independent scoring components (for example, multi-objective value functions).

Uniqueness theorem¶

Theorem (Shapley, 1953). There exists exactly one allocation rule \(\phi\) that satisfies efficiency, symmetry, the dummy-player axiom, and additivity. It is given by the formula above.

This uniqueness result is why SHAP (Lundberg & Lee, 2017) argues for Shapley-based additive feature attributions under local accuracy and missingness constraints: within the class of additive explanation models, the Shapley kernel is the unique choice satisfying a parallel set of desirable properties.

Connection to SHAP in machine learning¶

Lundberg & Lee (2017) cast feature attribution as a cooperative game where features are players and \(v(S)\) is the model output (or a transform) when only feature subset \(S\) is observed. Under local accuracy (attributions sum to the prediction difference from a baseline) and missingness (absent features do not contribute), the Shapley value yields the SHAP explanation.

Important distinction for nlp-shap:

  • SHAP names a family of explanation methods tied to Shapley allocations.

  • An estimand is the quantity your pipeline actually estimates. Monte Carlo averaging over sampled coalitions without Shapley weights targets the Banzhaf index, not the Shapley value — see Estimands: Shapley vs Banzhaf.

When Shapley values are appropriate¶

Shapley allocations are a strong default when you need:

  • Efficiency for reconciliation with total model utility

  • Fair treatment of symmetric players

  • Additive decomposition across independent utility components

  • Regulatory narratives that reference the standard game-theoretic fairness axioms

They are not always the right estimand when you want pivotal-player (voting-power) interpretations; Banzhaf indices emphasize how often a player swings a coalition from losing to winning. See Estimands: Shapley vs Banzhaf for when the two diverge.
