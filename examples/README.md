## Setup

Install the package, then open a notebook with Jupyter or VS Code.

From PyPI:

```bash
pip install nlp-shap
```

From a local clone:

```bash
git clone https://github.com/Pawlo77/nlp-shap
cd nlp-shap
make install
```

Use the project `.venv` as the notebook kernel when working from source.
