Examples¶

The examples/ directory contains Jupyter notebooks with runnable, end-to-end nlp-shap workflows. They complement the API reference.

Setup¶

Install the package:

pip install nlp-shap

From a local clone:

make install

Use the project .venv as the notebook kernel when working from source.

No notebooks are published yet. See the examples folder and examples/README.md for the catalog as notebooks are added.
