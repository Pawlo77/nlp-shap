## Installation

```bash
pip install nlp-shap
```

From source:

```bash
git clone https://github.com/Pawlo77/nlp-shap
cd nlp-shap
make install
```

Requires Python 3.12 and [uv](https://docs.astral.sh/uv/).

## Development

```bash
make install
make check
```

Run `make help` for other targets.
